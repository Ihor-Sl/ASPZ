# Character Frequency Analyzer in C

## Requirements

- A C compiler (such as GCC).
- A text file for input.

## Compilation

```
gcc -Wall char_freq.c -o char_freq
```

## Execution

```
./char_freq <file_path>
```

## Example Output

```

------------------------
    ' '   |    363
    'r'   |    85
    'e'   |    66
    'i'   |    56

```
